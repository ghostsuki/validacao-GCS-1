# Validação de Formulário com Modal - GCS-1

> Projeto prático desenvolvido para a disciplina de Gerência de Configuração de Software (GCS-1), com o objetivo de aplicar os conceitos de um fluxo de trabalho baseado em Git Flow, versionamento semântico e boas práticas de desenvolvimento em equipe.

---

### Funcionalidades

* **Modal Interativo:** Acesso ao formulário através de um modal que abre e fecha suavemente.
* **Validação de Formulário em Tempo Real:** Feedback instantâneo ao usuário sobre os requisitos de cada campo.
* **Campos Validados:**
    -   [✔] Nome (não pode estar vazio)
    -   [✔] E-mail (deve ter um formato válido)
    -   [✔] Senha (requer um número mínimo de caracteres)
    -   [✔] Situação de Trabalho (deve ser selecionada)
    -   [✔] Mensagem (não pode estar vazia)
* **Mensagens de Feedback:** Exibição de mensagens claras de sucesso ou erro após a submissão.

---

### Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web fundamentais:

* **HTML5:** Para a estrutura e semântica do conteúdo.
* **CSS3:** Para a estilização, layout e animações.
* **JavaScript (Vanilla JS):** Para toda a lógica de manipulação do DOM, interatividade do modal e validação do formulário.

---

### Como Executar

O projeto é totalmente front-end e não requer instalação de dependências.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/ghostsuki/validacao-GCS-1.git](https://github.com/ghostsuki/validacao-GCS-1.git)
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd validacao-GCS-1
    ```
3.  **Abra o arquivo `index.html`** no seu navegador de preferência.

E pronto! O formulário estará funcional.
